Este tutorial esta diseñado para ser una introducción independiente a C, incluso si nunca antes has programado. Sin embargo C++ es un lenguaje mas moderno, si no tienes claro si debes aprender C o C++, Se recomienda el tutorial de C++ en su lugar, el cual también es diseñado para personas que no han programado nunca. Sin embargo, si no desea algunas de las funciones avanzadas de C++ o simplemente desea aprender C en lugar de C++, entonces este es el sitio para ti.

Configuración: Encontrando un compilador

Las primera cosa que necesitas hacer, antes de empezar en C, es estar seguro que tienes un compilador. ¿Que es un compilador, te preguntaras? Un compilador convierte el programa que escribes en un ejecutable que tu computador actualmente puede entender y correr. Si estas tomando un curso, probablemente te proporcionan uno a través de tu escuela. Si estas empezando por tu propia cuenta, la mejor apuesta es usar Code::Blocks con MinGW. Si estas en Linux, puedes usar GCC, y su estas en Mac OS X, puedes usar XCode. Si aún no lo ha echo, continúe y configure un compilador, lo necesitará para el resto del tutorial.

Introducción a C

Cada programa C completo empieza dentro de una función llamada main. Una función es una simple colección de comandos o declaraciones que hacen algo. La función main siempre es llamada cuando el programa se empieza a ejecutar. Desde main, nosotros podemos llamar otras funciones, ya sean escritas por nosotros mismos o por otros o usar funciones incorporadas en lenguaje. Para acceder a las funciones estándar que vienen con tu compilador, necesitas incluir una cabecera con la directiva #include.
